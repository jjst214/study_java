인터페이스가 가지는 멤버
public 상수필드
public 추상메소드
public 디폴트메소드
public 정적메소드
private 메소드 : 디폴트 메소드 안에서만 호출 가능
* 디폴트와 정적 메소드 소스들의 중복 코드를 줄이기 위해서 사용
private 정적메소드 : 디폴트 메소드와 정적 메소드 안에서 호출 가능
*private 정적메소드는 디폴트 메소드에서도 사용가능하고
일반 private 메소드는 static 메소드에서 사용이 불가능함
